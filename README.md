# Fortemi

**Knowledge that endures.**

Fortemi builds AI-enhanced knowledge infrastructure with semantic search, automatic linking, and intelligent organization.

## About the Name

**Fortemi** (for-TEH-mee) blends Latin *fortis* (strong, resilient) with Japanese *emi* (harmony, beauty). Together: **resilient harmony** — the balance of strength and elegance in knowledge management.

## Projects

| Repository | Description |
|------------|-------------|
| [fortemi](https://github.com/Fortemi/fortemi) | Core knowledge base with hybrid search, semantic tagging, and AI integration |

## Features

- **Hybrid Search** — Full-text + semantic search with RRF fusion
- **Multilingual** — English, German, French, Spanish, Portuguese, Russian, CJK
- **Semantic Linking** — Automatic connections between related content
- **AI Integration** — MCP server for Claude and other AI assistants
- **Privacy-First** — End-to-end encryption for sensitive knowledge
